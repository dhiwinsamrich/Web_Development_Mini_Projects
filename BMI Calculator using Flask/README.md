# BMI Calculator using Flask

A simple calculator used to calculate the Body Mass Index (BMI).

## Technologies used:

- HTML
- CSS
- Python
- Flask

## How to use:

1. Fork and clone [this](https://github.com/dhiwinsamrich/Web_Development_Mini_Projects) project.
2. In your command prompt, navigate to **Web_Development_Mini_Projects/BMI Calculator using Flask**.
3. Run this app using the command `python app.py`.
4. Now, navigate to `http://127.0.0.1:5000/` in your web browser to use the calculator.
