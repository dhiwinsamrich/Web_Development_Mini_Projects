<h1>Password Generator App</h1>

<p>Password Generator App written in HTML, CSS, and JavaScript .</p>

### Use of the Project:

<p>This app helps to create strong random passwords which can be easily used during authentication.</p>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
</ul>
